# assets

- **Simple Calculator**
    
    **Concepts Used:** Variables, Operators, Functions, Conditions
    
    **Features:**
    
    - Take two numbers as input
    - Let the user choose an operator (+, −, ×, ÷)
    - Use a function to perform the calculation
    - Show an error message if dividing by 0
- **Student Grade Calculator**
    
    **Concepts Used:** Variables, Data Types, Operators, Conditions
    
    **Features:**
    
    - Input marks for 5 subjects
    - Calculate total and average
    - Assign grade using conditions:
        - 80+ → A
        - 70+ → B
        - 60+ → C
        - Below 60 → Fail
- **Even or Odd Checker**
    
    **Concepts Used:** Operators, Expressions, Conditions
    
    **Features:**
    
    - Input a number
    - Check whether it is even or odd
    - Bonus: Handle negative numbers correctly
- **Simple Login System (Without Database)**
    
    **Concepts Used:** Variables, Functions, Conditions
    
    **Features:**
    
    - Store a fixed username and password
    - Take user input
    - Check if credentials match
    - Show “Login Successful” or “Invalid Credentials”
- **Number Guessing Game**
    
    **Concepts Used:** Variables, Conditions, Functions (and loops if taught)
    
    **Features:**
    
    - Program generates a random number (1–10)
    - User tries to guess it
    - Show hints: “Too High”, “Too Low”, or “Correct!”
- **Simple ATM System**
    
    **Concepts Used:** Variables, Functions, Conditions, Operators
    
    **Features:**
    
    - Start with an initial balance
    - Menu options:
        - Check Balance
        - Deposit Money
        - Withdraw Money
    - Show error if balance is insufficient
- **Age Category Checker**
    
    **Concepts Used:** Conditions, Expressions
    
    **Features:**
    
    - Input age
    - Display category:
        - Child (<13)
        - Teen (13–19)
        - Adult (20–59)
        - Senior (60+)
- **Electricity Bill Calculator**
    
    **Concepts Used:** Operators, Conditions
    
    **Features:**
    
    - Input electricity units used
    - Calculate bill based on rules:
        - First 50 units → 5 per unit
        - Next 100 units → 8 per unit
        - Remaining units → 10 per unit
- **Multiplication Table Generator**
    
    **Concepts Used:** Loops, Operators
    
    **Features:**
    
    - Input a number
    - Print its multiplication table
